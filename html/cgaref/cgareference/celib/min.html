min

Syntax

float    min(a, b)

string   min(a, b)

float[]  min(a, b)

string[] min(a, b)

Parameters

  • a (float, string, float[], string[])
  • b (same type as a)

Returns

Returns minimum of a and b.

Related

  • max function

Examples

min(10, -10)
# result = -10	

min("abc", "def")
# result = "abc"
