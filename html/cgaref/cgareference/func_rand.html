rand function

Syntax

float rand()

float rand(max)

float rand(min, max)

Parameters

  • min (float)
  • max (float)

Returns

A random value in [min, max]. The value depends on the current shape's seedian.

Description

This function returns a random value in the selected range. Defaults are 0 for min and 1 for max.
