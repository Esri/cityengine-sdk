colorHexToO

Synopsis

string colorHexToO(string hex)

Parameters

  • hex
    hex color string in the format "#rrggbboo"

Returns

returns O value in [0,1] from hex color string in the format "#rrggbboo" where "oo" are the opacity values

This function extracts the opacity value from a hex color string and converts it to a value in [0,1]. Returns 0 if hex string is too short.
