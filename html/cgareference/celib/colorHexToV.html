colorHexToV

Synopsis

string colorHexToV(string hex)

Parameters

  • hex
    hex color string in the format "#rrggbb" or "#rrggbboo"

Returns

returns V value in [0,1] after converting hex color to HSV.

This function converts a hex color string to HSV and returns the V value.
