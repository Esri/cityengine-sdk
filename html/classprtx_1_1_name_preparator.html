Detailed Description

NamePreparator interface used to sanitize object names.

Member Enumeration Documentation

enum prtx::NamePreparator::Entity

These entities control the legalize and uniquify behavior. For example, a file name has very likely a different set of legal characters than a mesh name.

Enumerator
