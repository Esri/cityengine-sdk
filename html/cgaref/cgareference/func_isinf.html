isinf ("is infinite") function

Syntax

bool   isinf(value)

bool[] isinf(values)

Parameters

  • value (float)
  • values (float[])

Returns

true if value is infinite, false otherwise.

For an array values a new array is returned containing the result of each element.

Related

  • isnan function

Examples

isinf(1)
# result = false

isinf(1/0)
# result = true

isinf(ln(0))
# result = true

isinf([1, 1/0])
# result = [false, true]
