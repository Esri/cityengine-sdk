colorHSVToHex

Synopsis

string colorHSVToHex(float hue, float sat, float val)

Parameters

  • hue
    H value in [0,360] degrees of HSV color
  • sat
    S value in [0,1] of HSV color
  • val
    V value in [0,1] of HSV color

Returns

returns hex color string (format: "#rrggbb") for HSV color.

This function converts an HSV color to a hex color string. It converts negative degrees or degrees greater than 360 for hue to the range [0,360]. It clamps sat and val to the range [0,1].
