sel function

Syntax

selector sel(selString)

Parameters

  • selString (string)

Returns

Selector with the value of selString.

Description

This conversion function allows to cast the selString to a selector.

Examples

colchannel(index) = "material.color."+index

A -->
    set(sel(colchannel("r")),0.5)
