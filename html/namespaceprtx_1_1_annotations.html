Detailed Description

Supported annotation types for encoder options or encoder-supported initial shape attributes. Typically these annotations are used to programmatically construct a user interface from an EncoderInfo instance or to make certain initial shape attributes mandatory, for example.

The documentation uses "option", "option attribute", "initial shape attribute" synonymously.
