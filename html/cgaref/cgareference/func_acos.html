acos function

Syntax

float acos(x)

Parameters

  • x (float)

Returns

The arc cosine of x in degrees. This value is in the range [0, 180].
If x is outside [-1,1], nan is generated.

Description

The acos function calculates the arc cosine of x, i.e. the value whose cosine is x.
