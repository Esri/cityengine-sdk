max

Synopsis

float max(float a, float b)

Parameters

  • a
    float or string (b must have the same type).
  • b
    float or string (a must have the same type).

Returns

returns maximum of a and b.

This function finds the maximum of a and b. It also works if a and b are strings.

Related

  • min function

Examples

example.1
	max(10, -10)
	# result = 10	

example.2
	max("abc", "def")
	# result = "def"
