min

Synopsis

float min(float a, float b)

Parameters

  • a
    float or string (b must have the same type).
  • b
    float or string (a must have the same type).

Returns

returns minimum of a and b.

This function finds the minimum of a and b. It also works if a and b are strings.

Related

  • max function

Examples

example.1
	min(10, -10)
	# result = -10	

example.2
	min("abc", "def")
	# result = "abc"
