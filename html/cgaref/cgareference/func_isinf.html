isinf ("is infinite")

Synopsis

bool isinf(float value)

Returns

true if value is infinite, false otherwise

This function checks if value is an infinite value.

Related

  • isnan function

Examples


example.1
	isinf(1/0)
	# result = true
	
example.2
	isinf(ln(0))
	# result = true
