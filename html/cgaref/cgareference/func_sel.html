sel

Synopsis

float sel(string selstring)

Returns

Selector with the value of inputstring.

This conversion function allows to cast the inputstring to a selector.

Examples

	colchannel(index) = "material.color."+index
    
	A -->
		set(sel(colchannel("r")),0.5)
