clamp

Syntax

float    clamp(x, min, max)

string   clamp(x, min, max)

float[]  clamp(x, min, max)

string[] clamp(x, min, max)

Parameters

  • x (float, string, float[], string[])
  • min (same type as x)
  • max (same type as x)

Returns

Returns min if x is less than min, max if x is greater than max, and x otherwise.

Description

This function clamps x to the range [min, max].

Related

  • abs function
  • ceil function
  • floor function
  • rint function

Examples

clamp(1.5, 0, 1)
# result = 1

clamp(-11, -10, 10)
# result = -10
