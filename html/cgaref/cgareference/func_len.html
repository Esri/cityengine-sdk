len

Synopsis

float len(string inputstring)

Returns

Length of inputstring.

This function returns the number of characters of the inputstring.

Related

  • count function
  • find function
  • substring function

Examples

example.1
	len("cityengine")
	# result = 10

example.2
	len("abc")
	# result = 3
