colorRGBOToHex

Synopsis

string colorRGBOToHex(float r, float g, float b, float o)

Parameters

  • r
    R value in [0,1] of RGB color.
  • g
    G value in [0,1] of RGB color.
  • b
    B value in [0,1] of RGB color
  • o
    opacity value in [0,1]

Returns

returns hex color string (format: "#rrggbboo") for RGB color in [0,1] with opacity in [0,1].

This function converts an RGB color with opacity to a hex color string. It clamps r, g, b, and o to the range [0,1].
