min

Syntax

float min(a, b)

Parameters

  • a (float, string)
  • b (float, string)
    Must have the same type as a.

Returns

Returns minimum of a and b.

Description

This function finds the minimum of a and b. It also works if a and b are strings.

Related

  • max function

Examples

example.1
	min(10, -10)
	# result = -10	

example.2
	min("abc", "def")
	# result = "abc"
