colorHexToH

Synopsis

string colorHexToH(string hex)

Parameters

  • hex
    hex color string in the format "#rrggbb" or "#rrggbboo"

Returns

returns H value in [0,360] degrees after converting hex color to HSV.

This function converts a hex color string to HSV and returns the H value.
