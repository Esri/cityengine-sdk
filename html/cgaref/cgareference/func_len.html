len function

Syntax

float len(inputString)

Parameters

  • inputString (string)

Returns

Length of inputString.

Description

This function returns the number of characters of the inputString.

Related

  • count function
  • find function
  • substring function

Examples

example.1
	len("cityengine")
	# result = 10

example.2
	len("abc")
	# result = 3
