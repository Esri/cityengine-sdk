max

Syntax

float    max(a, b)

string   max(a, b)

float[]  max(a, b)

string[] max(a, b)

Parameters

  • a (float, string, float[], string[])
  • b (same type as a)

Returns

Returns maximum of a and b.

Related

  • min function

Examples

max(10, -10)
# result = 10	

max("abc", "def")
# result = "def"
