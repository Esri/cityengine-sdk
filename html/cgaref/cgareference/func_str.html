str function

Syntax

string str(value)

Parameters

  • value (float, bool, string, float[], bool[], string[])
    Value to convert.

Returns

String calculated from value.

Description

This function converts value to the corresponding string value.

Related

  • float function
  • bool function
  • floatArray function
  • boolArray function
  • stringArray function

Examples

str(1.69)
# result = "1.69"	

str(1.69 - 69/100)
# result = "1"	

str(001)
# result = "1"	

str(true)
# result = "true"
