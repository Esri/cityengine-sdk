Detailed Description

Returns a summary for all reports:

  • cardinality: <key>_n
  • average: <key>_avg
  • sum: <key>_sum
  • minimum: <key>_min
  • maximum: <key>_max

For booleans the summary is computed by using the binary values. For strings the sum and avg are the most frequent values while min/max are computed lexicographically.
