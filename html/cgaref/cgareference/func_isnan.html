isnan ("is not a number") function

Syntax

bool isnan(value)

Parameters

  • value (float)

Returns

true if value is not a number, false otherwise.

Description

This function checks if value is not a number.

Related

  • isinf function

Examples

example.1
	isnan(1)
	# result = false	

example.2
	isnan(ln(-1))
	# result = true

example.3
	isnan(float("a"))
	# result = true
