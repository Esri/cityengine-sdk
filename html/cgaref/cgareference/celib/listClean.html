listClean

Syntax

string listClean(stringList)

Parameters

  • stringList (string)

Returns

Returns cleaned stringList.

Description

This function checks if the stringList has the correct ";" at the end. Additionally, empty items (';;') are removed.

Note: String lists are a series of strings stored inside one string. The elements are separated by a semicolon (";"). Each item must be followed by a semicolon, also the last one! The data type is "string", thus it is not an array as used in other scripting languages.

Examples

listClean(";;;;;my;city;rules;;your;;city;;")
# result = "my;city;rules;your;city;"

listClean(";;cityengine;;;rules")
# result = "cityengine;rules;"
