label operation

Synopsis

label(label)

Parameters

  • label (string)
    A non-empty label.

Creates a labeled shape. This shape can be referenced to in various labeled context related queries, for example occlusion queries on selected occluder shapes.

Example: Occlusion queries

The label operation makes the current shape available for intra and inter occlusion queries.

Labeled occlusion queries

Labeled occlusion queries are performed on occluder shapes with that label only.

Unlabeled occlusion queries

Unlabeled occlusion queries are performed on occluder shapes with any label.

Additionally, for leaf shapes and shapes where a component split or offset has been applied, (unlabeled) occluder shapes are automatically generated. In most cases this is sufficient, and there is no need to use the label operation.
