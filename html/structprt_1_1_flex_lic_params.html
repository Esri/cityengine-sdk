const char* prt::FlexLicParams::mActLibPath

The absolute path to the activation library (e.g. flexnet_prt.dll on windows).

const char* prt::FlexLicParams::mFeature

For standard CE installations: The license feature string, i.e. one of

  • CityEngBas (basic network license)
  • CityEngBasFx (basic node-locked license)
  • CityEngAdv (advanced network license)
  • CityEngAdvFx (advanced node-locked license)
const char* prt::FlexLicParams::mHostName

Host name of the license server (empty for node-locked licensing). For the default flexnet port 27000, the syntax is

27000@myserver.example.com
