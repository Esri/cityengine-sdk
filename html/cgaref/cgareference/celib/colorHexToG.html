colorHexToG

Syntax

float colorHexToG(hexValue)

Parameters

  • hexValue (string)
    Hexadecimal color string in the format "#rrggbb" or "#rrggbboo"

Returns

Returns G value in [0,1] after converting hexadecimal color to RGB.

Description

This function converts a hexadecimal color string to RGB and returns the G value.

# result = 0.5019607843137256
