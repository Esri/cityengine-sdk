colorHexToS

Synopsis

string colorHexToS(string hex)

Parameters

  • hex
    hex color string in the format "#rrggbb" or "#rrggbboo"

Returns

returns S value in [0,1] after converting hex color to HSV.

This function converts a hex color string to HSV and returns the S value.
