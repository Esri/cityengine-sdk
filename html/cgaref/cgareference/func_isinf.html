isinf ("is infinite") function

Syntax

bool isinf(value)

Parameters

  • value (float)

Returns

true if value is infinite, false otherwise

Description

This function checks if value is an infinite value.

Related

  • isnan function

Examples

isinf(1/0)
# result = true

isinf(ln(0))
# result = true
