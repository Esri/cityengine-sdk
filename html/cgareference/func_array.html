array initialization function

Synopsis

float[] [value1, value2, ...]

bool[] [value1, value2, ...]

string[] [value1, value2, ...]

Parameters

  • value1, value2, ... (float, float[] or string, string[] or bool, bool[])
    Array element values. At least one element must be specified.

Returns

An array containing all elements in the specified order.

This function initializes an array by the contatenation of primitive values (float, string, bool) and/or arrays (float[], string[], bool[]).

Example

const a = [1, 2, 1] // (3)[1,2,1]

const b = [0, a, 0] // (5)[0,1,2,1,0]
