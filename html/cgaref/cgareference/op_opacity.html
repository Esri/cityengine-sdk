opacity operation

Synopsis

opacity(f)

Parameters

  • f (float)
    Opacity value. 1 is fully opaque, 0 is fully transparent.

The opacity operation sets the opacity factor of the current shape's material.

Note: this command has the same effect as a call to set(material.opacity, f), it is a shortcut for convenience.
