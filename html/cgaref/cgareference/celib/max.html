max

Syntax

float max(a, b)

Parameters

  • a (float, string)
  • b (float, string)
    Must have the same type as a.

Returns

Returns maximum of a and b.

Description

This function finds the maximum of a and b. It also works if a and b are strings.

Related

  • min function

Examples

max(10, -10)
# result = 10	

max("abc", "def")
# result = "def"
