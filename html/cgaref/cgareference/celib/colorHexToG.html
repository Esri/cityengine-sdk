colorHexToG

Synopsis

string colorHexToG(string hex)

Parameters

  • hex
    hex color string in the format "#rrggbb" or "#rrggbboo"

Returns

returns G value in [0,1] after converting hex color to RGB.

This function converts a hex color string to RGB and returns the G value.
