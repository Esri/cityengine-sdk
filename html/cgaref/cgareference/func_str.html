str function

Synopsis

string str(string value)

string str(float value)

string str(bool value)

Returns

String calculated from value.

This function converts value to the corresponding string value.

Related

  • float function
  • bool function

Examples

example.1
	str(1.69)
	# result = "1.69"	

example.2
	str(1.69 - 69/100)
	# result = "1"	
    
example.3
	str(001)
	# result = "1"	
    
example.4
	str(true)
	# result = "true"
